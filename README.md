#### System Compatibility
- **Minimum macOS:** 10.13 (High Sierra)
- **Processor support:** Intel (64-bit) and Apple Silicon (Universal 2 binary)
- **Storage requirements:** 100MB for application, 500MB+ recommended for index storage

#### Installation Process

**Standard Installation:**
1. Download the ProFind installation package
2. Mount the .dmg disk image
3. Drag ProFind to your Applications folder
4. Launch from Applications folder or Launchpad

**First Launch Configuration:**
1. Grant necessary permissions when prompted (Full Disk Access for content indexing)
2. Configure initial indexing preferences
3. Select folders to include in search index
4. Set up exclusion filters for system folders if desired
